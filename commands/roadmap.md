# Roadmap

Start or continue a personalized learning path based on your goals, current skills, and time constraints. Get a detailed learning sequence with phases, checkpoints, and milestones.

## Usage

```
/roadmap start                    # Begin a new learning path
/roadmap continue                 # Continue current path
/roadmap <path-name>              # Load specific roadmap
/roadmap plan <goal>              # Create custom learning plan
/roadmap timeline <duration>      # Adapt timeline to availability
```

## Examples

```
/roadmap start
# Choose from 65+ available paths and begin learning

/roadmap frontend-developer
# Load Frontend Developer roadmap

/roadmap plan full-stack-in-6-months
# Create custom plan for specific goal and timeframe

/roadmap timeline part-time
# Adjust recommended path for 10-15 hours/week
```

## Your Learning Path

The plugin creates:

### Phase-Based Structure
```
PHASE 1: Fundamentals (Weeks 1-4)
├─ Core topics and basics
├─ First projects
├─ Checkpoint assessment
└─ Prerequisites for Phase 2

PHASE 2: Intermediate (Weeks 5-12)
├─ Advanced concepts
├─ Framework/tool mastery
├─ Real-world patterns
└─ Medium-complexity projects

PHASE 3: Advanced (Weeks 13+)
├─ Specialization topics
├─ Performance optimization
├─ Architecture patterns
└─ Portfolio-ready projects
```

### Learning Sequence
- Proper prerequisite ordering
- Progressive difficulty increase
- Optimal skill combinations
- Checkpoint milestones
- Project recommendations per phase

### Resource Bundles
- Curated materials for each topic
- Multiple learning styles covered
- Free and affordable options
- Time estimates per resource

### Timeline Estimates
- Full-time (40+ hours/week): 4-6 months
- Part-time (15-20 hours/week): 9-12 months
- Very part-time (<10 hours/week): 12-24 months
- Custom based on your pace

## Adjusting Your Path

```
/roadmap adjust pace slower    # Extend timeline
/roadmap adjust pace faster    # Accelerate (if sustainable)
/roadmap adjust focus <skill>  # Add specialization
/roadmap skip <topic>          # Skip topic you know
/roadmap add-project           # Add practice project
/roadmap extend <phase>        # Spend more time on phase
```

## Tips for Success

3. **Track progress**: Celebrate milestones

6. **Review often**: Spaced repetition helps retention
