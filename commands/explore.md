# Explore

Discover and explore 65+ developer roadmaps covering all major technology domains, career paths, and specializations.

## Usage

```
/explore                    # Show all roadmap categories
/explore <category>         # Explore specific category (frontend, backend, etc)
/explore <specific-roadmap> # Deep dive into specific roadmap
/explore search <keyword>   # Search roadmaps by technology
/explore compare <path1> <path2> # Compare two paths
```

## Examples

```
/explore
# Shows all 65+ available roadmaps organized by category

/explore frontend
# Shows all frontend-related roadmaps and specializations

/explore react
# Deep dive into React roadmap with topics, resources, and projects

/explore search typescript
# Find all roadmaps related to TypeScript

/explore compare backend devops
# Compare Backend Developer vs DevOps Engineer paths
```
